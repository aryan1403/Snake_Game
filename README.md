# Snake_Game
A JAVA GUI Snake Game with AI to play Automatically

This Project is under Development (AI yet to be Implemented)

A GUI Based Application written in Java Swing. 
I will be Updating This Code in the Future for improved Code and better functionalities.

## Language
  This project is made in Pure Java Swing ;)
